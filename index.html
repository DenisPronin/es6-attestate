Фичи ES6

Объявление переменной с let

Область видимости переменной let — блок {}

ES5


var apples = 5;
if (true) {
    var apples = 10;
    alert(apples); // 10
}
alert(apples); // 10

ES-2015


let apples = 5; // (*)
if (true) {
    let apples = 10;
    alert(apples); // 10
}
alert(apples); // 5
